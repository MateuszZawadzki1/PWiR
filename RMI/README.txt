OPIS APLIKACJI:
Aplikacja to chat działający w sieci lokalnej przy pomocy RMI.
Umozliwia komunikacje miedzy uzytkownikami za pomoca wysylania i odbierania wiadomosci.

INSTRUKCJA URUCHOMIENIA(Terminal|Miejsce plików):
1. Kompilacja wszystkich klas, wpisz: javac *
2. Rejstracja zdalnych obiektów, wpisz: rmiregistry
3. W nowym oknie terminala, wlacz serwer za pomocą komendy: java ChatServerDriver
4. Włącz nowe okno terminala, aby zalogować się jako uzytkownik do czatu, wpisując: java ChatClientDriver "nazwa uzytkownika"
5. Powtórz punkt numer 4 dla kazdej kolejnego uzytkownika
